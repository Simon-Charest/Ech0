# Hampton Bay Harbor Breeze Litex UC9051T (DIP6)
- Filetype: Flipper SubGhz Key File
- Version: 1
- Frequency: 302757000 (302.757 MHz)
- Preset: FuriHalSubGhzPresetOok650Async
- Protocol: Holtek_HT12X
- Bit: 12
- TE (Transmit Enable): 333-336
